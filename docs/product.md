# Product Overview

## 产品定义

AI 送报员是一个面向个人用户的 AI 个人情报助理。

它解决的问题不是“今天互联网发生了什么”，而是：

“今天对我最有用、最值得知道、最不该错过的内容是什么？”

## MVP 核心原则

- 目标导向，不做泛资讯堆砌
- 相关性优先，但保留可控探索
- 邮件日报是第一交付形态
- 用行为信号替代高频打扰
- 先做稳定流水线，不引入重型 agent 编排

## MVP 关键体验

- 每天固定时间送达
- 内容少而精
- 明确说明“为什么推给你”
- 支持轻反馈
- 允许少量意外收获

## 日报分区

- `must_read` 今天必看
- `focus_updates` 关注方向新进展
- `worth_saving` 值得收藏
- `surprise` 今天的意外收获

## MVP 成功标准

- 用户愿意打开邮件
- 用户觉得这封日报节省了时间
- 用户愿意连续几天继续接收
